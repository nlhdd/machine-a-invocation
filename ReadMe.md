# 🐶 doggoSummoner()

Summon your doggo in one click.

L'utilisateur entre son prénom, choisit un chien, et l'invoque.

The user enters their name, picks a dog, and summons it.

---

## 🇫🇷 Français

### Aperçu

`doggoSummoner()` est une page web construite avec **HTML, CSS et JavaScript** (sans framework). Elle met en pratique la manipulation du DOM, la gestion d'événements, l'animation de sprites en CSS, et le basculement entre deux écrans (formulaire / résultat).

### Fonctionnalités

- Formulaire d'invocation avec saisie du prénom et choix du chien
- Quatre chiens (Golden Retriever, Akita, Saint-Bernard, Husky)
- Écran de résultat 

### Stack

- **HTML5** 
- **CSS3**
- **JavaScript** 

### Lancer le projet

Ouvrir `index.html` dans un navigateur.

---

## 🇬🇧 English
